# Insta bot
### Bot para ganhar seguidores através de troca de follows
___

### Configurar ambiente
 - Definir variáveis de ambiente `INSTA_LOGIN` e `INSTA_SENHA`.
 - `pip install pipenv`.
---
### Rodar testes
 - `pipenv install --dev`
 - `pipenv shell`
 - `pytest`
---
#### Rodar projeto:
 - `pipenv shell`.
 - `python main.py`.
